# PCの設定

## ディレクトリって何？

ターミナルを開いて`ls`と入力すると青字で書かれた単語が表示される。これをディレクトリと言い、ファイルやディレクトリを格納する場所を示す。`cd (ディレクトリ名)`を入力することでディレクトリの移動ができる。`pwd`で現在の位置を確認することができる。このディレクトリの位置のことをPATHといい、`cd (パス)`を入力することで参照したいディレクトリまで一気に移動することができる。ホームディレクトリに戻りたい場合は`cd`,一つ前のディレクトリに戻りたい場合は`cd ../`と入力する。

## ディレクトリの編集

ディレクトリ内のファイルを開いて編集したい場合,以下のコマンドで編集画面を開くことができる。

```shell
vi (ファイル名)
```

新たにディレクトリを作成する場合

```shell
mkdir (ディレクトリ名)
```
ディレクトリやファイルを管理する際にディレクトリの階層構造を整理しておくことは今後の研究活動を進める上で非常に重要となる。ここで高分子グループの研究で出てくるディレクトリ整理の一例を紹介する。高分子グループで出てくるファイル類は主に4つに分けられる。

1. 物性の調整、解析に必要なC++のプログラム
2. 1のプログラムを動かすのに必要なアプリ
3. グラフ化した計算結果やデータ
4. 研究に必要な文献

これらを1-bin,2-apl,3-research,4-articleに分けた。
