===================== GROMACSチュートリアル¶

このチュートリアルでは、水とイオンが入ったボックス内のタンパク質（lysozyme）を扱う。

.. important:: チュートリアルではGROMACS 2018.xシリーズを使用するため、手元のバージョンを確認しておく。

目次¶

このチュートリアルの目標は、MDシミュレーションの流れを理解しながらGROMACSの基本的な使い方を習得することである。 そのため、シミュレーションに必要な設定ファイルの内容には触れていない。 ファイルの詳細については【補足】にまとめているので、より理解を深めるために参照してほしい。
